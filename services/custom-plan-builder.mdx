The Custom Plan Builder allows you to create a tailored web solution by selecting exactly the features you need. Choose from pre-configured website types or build from scratch.

## How It Works

<Steps>
  <Step title="Choose Website Category">
    Select from 7 website types (E-commerce, Blog, School, Portfolio, Business, SaaS, or Other) to get recommended features
  </Step>
  
  <Step title="Review Recommended Features">
    Each category includes carefully selected features with estimated pricing
  </Step>
  
  <Step title="Customize Your Selection">
    Add or remove features from 7 categories to match your exact requirements
  </Step>
  
  <Step title="Get Instant Quote">
    See real-time pricing as you build your package
  </Step>
  
  <Step title="Submit Your Request">
    Complete the form to receive a detailed proposal
  </Step>
</Steps>

## User Interface

The Custom Plan Builder provides an intuitive experience:

<Tabs>
  <Tab title="Website Type Selection">
    Choose from 7 pre-configured website types with estimated pricing. Each type displays:
    - Category icon
    - Estimated total cost
    - Number of recommended features
    - Brief description
  </Tab>
  
  <Tab title="Feature Selection">
    Browse features organized by category. Each feature shows:
    - Feature name and price
    - Detailed description with bullet points
    - Recommended badge (if applicable to selected website type)
    - Visual selection indicator
  </Tab>
  
  <Tab title="Summary Panel">
    Sticky sidebar (desktop) or bottom bar (mobile) showing:
    - Selected website type
    - Features grouped by category
    - Individual and category totals
    - Overall total investment
    - Continue button to submit
  </Tab>
  
  <Tab title="Request Form">
    Modal form collecting:
    - Full name (required)
    - Email address (required)
    - Phone number (required)
    - Company name (optional)
    - Additional notes (optional)
  </Tab>
</Tabs>

## Best Practices

<Card title="Tips for Using the Custom Plan Builder" icon="lightbulb">
  <AccordionGroup>
    <Accordion title="Start with a Website Type">
      Select a website category first to get smart feature recommendations based on your industry
    </Accordion>
    
    <Accordion title="Review Core Features">
      Core features (Responsive Design, Basic SEO) are automatically included for most website types
    </Accordion>
    
    <Accordion title="Consider Hosting Needs">
      Choose hosting based on expected traffic and resource requirements. VPS options are recommended for high-traffic or resource-intensive sites
    </Accordion>
    
    <Accordion title="Plan for Growth">
      Select features that support your growth trajectory. It's easier to build scalability in from the start
    </Accordion>
    
    <Accordion title="Note Annual Billing">
      Some features (hosting solutions) use annual billing. Check for the "(Annual)" indicator
    </Accordion>
  </AccordionGroup>
</Card>
